Least Squares Solution¶

$$ \hat w = \underset{w}{\mathrm{argmin}}\space || X w - y||^2 \Rightarrow \frac{\partial}{\partial w} || X w - y||^2 \stackrel{!}{=} 0 $$$$ \begin{aligned} 0 &= \frac{\partial}{\partial w} (X w - y)^T (X w - y) \\ 0 &= \frac{\partial}{\partial w} w^T X^T X w - w^T X^T y - y^T X w + y^2 \\ 0 &= (X^T X w)^T + (w^T X^T X) - (X^T y)^T - (y^T X) \\ 2(X^T X w)^T &= 2(X^T y)^T \\ X^T X w &= X^T y \\ w &= (X^T X)^{-1} X^T y \\ \end{aligned} $$

d = 8

# Create target matrix X
X = []
for i in range(d):
    X.append(x**i)
X = np.stack(X).T

# Apply least squares
w = np.linalg.inv(X.T @ X) @ X.T @ y
print("w:", w)

# Make predictions according to learned parameter w
y_pred = X @ w

Regularized Least Squares Solution (L2 Loss)¶

$$ \hat w = \underset{w}{\mathrm{argmin}}\space || X w - y||^2 + \lambda||w||^2 \Rightarrow \frac{\partial}{\partial w} || X w - y||^2 + \lambda||w||^2 \stackrel{!}{=} 0 $$$$ \begin{aligned} 0 &= \frac{\partial}{\partial w} (X w - y)^T (X w - y) + \lambda w^T w \\ 0 &= \frac{\partial}{\partial w} w^T X^T X w - w^T X^T y - y^T X w + y^2 + \lambda w^T w \\ 0 &= \frac{\partial}{\partial w} w^T X^T X w - w^T X^T y - y^T X w + y^2 + \lambda w^T I_d w \\ 0 &= (X^T X w)^T + (w^T X^T X) - (X^T y)^T - (y^T X) + (\lambda I_d w)^T + (\lambda w^T I_d) \\ 2(X^T X w)^T + 2 (\lambda I_d w)^T &= 2(X^T y)^T \\ X^T X w + \lambda I_d w &= X^T y \\ w &= (X^T X + \lambda I_d)^{-1} X^T y \\ \end{aligned} $$

It can be seen that regularization has effect on the bias. Generally it is not advised to apply regularization to bias terms. Let's do the calculations again without applying regularization to the bias.

In [7]:
d = 8
lambd = 2

# Create target matrix X
X = []
for i in range(d):
    X.append(x**i)
X = np.stack(X).T

# Apply least squares
C = lambd*np.eye(d)
C[0,0] = 0.0  # SET VALUE TO ZERO FOR EXCLUDING BIAS TERM (because w[0] is the bias)
w = np.linalg.inv(X.T @ X + C) @ X.T @ y
print("w:", w)

# Make predictions according to learned parameter w
y_pred = X @ w

# Plot
_=plt.plot(x,y, ".")
_=plt.plot(x,y_pred)
_=plt.xlabel("x")
_=plt.ylabel("y")
